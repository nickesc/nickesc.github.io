Help and Usage
This site is know to have formatting issues on Firefox; all site functionality remains available

Type commands from the list below into the terminal.
Use UP and DOWN arrow keys to scroll through command history
Use CRTL+C to clear the command line

command usage
cright show copyright information
help show the help screen
open <tab> open a tab; used with tabs in the toolbar
power toggle power; stop or start the terminal
<projectCommand> redirect to a project; the <projectCommand> is the project's name without spaces
<projectCommand> source redirect to a project's source
<projectCommand> img redirect to a project's logo image
resume redirect to my resume
source redirect to site source
firefox open the Firefox-specific formatting warning
start start the terminal
stop stop the terminal
